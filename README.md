# Echo Web Application

A simple peer-graded assignment web app. The page accepts user input and echoes the submitted message back to the screen.

## Files

- `index.html` - page structure
- `style.css` - responsive styling
- `script.js` - input handling and echo behavior

## Run Locally

Open `index.html` in a browser.

## Deploy

This project can be hosted with GitHub Pages, Netlify, or Vercel because it is a static web app.
